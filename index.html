iD3 | source code

iD3 is an integrated development enviroment for data driven documents (D3.js). It's a tool to get users familiarized with D3 and have their projects ready to upload to the browser.
It's built in electron, with React, and Redux. It features controlled components, and a container-view design.

Dollar Shirt Club | source code

Fast and scalable e-commerce site built with React and Redux. Other features involve unit testing and test coverage with Jest and Enzyme.
Sass (pre-processor) is also used for styling to follow dry principles and make code more maintainable.
